CAPAS DEL PROYECTO

1. Vista
   Carpeta: frontend/
   Archivos:
   - index.html
   - app.js
   - styles.css

2. Controladores
   Carpeta: backend/app/controllers/
   Archivos:
   - auth.py
   - users.py
   - products.py
   - orders.py
   - inventory.py
   - reports.py

3. Modelo / Base de datos
   Archivos:
   - backend/app/models.py
   - backend/app/database.py
   - config/database.sql

4. Validaciones
   Archivo:
   - backend/app/schemas.py

5. Seguridad
   Archivos:
   - backend/app/security.py
   - backend/app/dependencies.py

6. Datos iniciales
   Archivo:
   - backend/app/seed.py


TECNOLOGIA

- Backend: FastAPI
- Frontend: Vue desde CDN, sin package.json
- Base de datos: SQL Server
- ORM: SQLAlchemy
- Login: JWT
